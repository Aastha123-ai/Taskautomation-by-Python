# JPG File Organizer

## Overview

JPG File Organizer is a Python automation script that automatically moves all `.jpg` image files from a source folder to a destination folder.

The script checks whether the source and destination folders exist, creates the destination folder if necessary, and transfers image files while displaying progress in the terminal.

## Features

* Automatically detects `.jpg` files
* Creates destination folder if it does not exist
* Moves files safely using Python's shutil module
* Displays moved files in real time
* Simple and lightweight automation tool

## Technologies Used

* Python
* os Module
* shutil Module

## How It Works

1. Select a source folder.
2. Select a destination folder.
3. Run the script.
4. All `.jpg` files are moved automatically.

## Sample Use Case

Source Folder:
C:\Users\Public

Destination Folder:
C:\Users\daast

The script scans the source directory and moves all JPG images to the destination directory.
